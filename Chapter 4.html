Programming Techniques

Programming Constructs

Sequence: A set of instructions or data
Iteration: A repeat or loop of a section of a sequence
Branching: Going from one section of a sequence to another
Selection: Similar to branching, a decision or split in the sequence
Recursion: The recursion of a procedure or definition

Variables & Parameters

When writing any code it is quite likely that you will use variables, now it is important to understand the difference between global and local variables as it can have a big effect on your code. Global variables are shared across your entire file, so if you use that name again it will use that same variable value however with local variables we can use the same name in different section of code it will have a different value.

When passing a parameter (a variable inputted in a subroutine) there are two ways of doing it. Pass it by value: Just move the value of the variable and passing by Reference: Pass the adress of the memory location where the parameter is stored.

Integrated Development Enviroments

IDE's have a vast array of tools built into them to help develop and improve code. These include things such as:
Compiler: Compiles the code for you so you dont have to have a seperate compiler program
Debugger: Checks the code for you and points out any obvious mistakes that you need to change to remove bugs
Error Checking: If there is an error in our code when you compile it, this should point out where the error is coming from
Autocomplete functions: When you type half a word it will give suggestions for the rest of your word or phrase
Colour Coding: It will colour different aspects of your code depending on its type eg in python comments are green
